# RENEWABLE_RESOURCE_STOCK_07_TO_24

🌱 Renewable Energy Stock Analysis (2007–2024)

📌 Overview
This SQL project analyzes renewable energy stock data from 2007–2024. It includes basic and intermediate SQL queries to explore energy resources, generation trends, resource contributions, and yearly performance.

🛠 Technologies Used
MySQL
SQL

📊 Key Analyses
• Resource-wise record count
• Unique energy resources
• Yearly data trends
• Total and average generation
• Highest and lowest generation values
• Resource contribution percentages
• Top-performing energy resources
• Generation categorization (High, Medium, Low)
